Project Overview:

This project is an inventory management system for a clothing store. It includes features to manage products, customers, and orders. The system allows you to add, update, and delete products, manage customer information, and track orders.

Files Included:

 1. inventory.py: Manages the inventory of products, allowing you to add, update, and delete products.
 2. customers.py: Handles customer information, including adding and updating customer details.
 3. orders.py: Manages orders placed by customers, including adding and updating orders.
 4. database.db: SQLite database file used to store product, customer, and order records.
 5. main.py: Provides a command-line interface to interact with the inventory, customers, and orders.

Usage:

 1. Run main.py to start the program.
 2. Follow the on-screen prompts to manage inventory, customers, and orders.

توضیحات پروژه:

این پروژه یک سیستم مدیریت موجودی برای فروشگاه لباس است. این سیستم شامل ویژگی‌هایی برای مدیریت محصولات، مشتریان و سفارشات است. شما می‌توانید محصولات را اضافه، به‌روزرسانی و حذف کنید، اطلاعات مشتریان را مدیریت کنید و سفارشات را پیگیری کنید.

فایل‌های موجود:

 1. inventory.py: مدیریت موجودی محصولات، از جمله اضافه، به‌روزرسانی و حذف محصولات.
 2. customers.py: مدیریت اطلاعات مشتریان، از جمله اضافه و به‌روزرسانی جزئیات مشتری.
 3. orders.py: مدیریت سفارشات مشتریان، شامل اضافه و به‌روزرسانی سفارشات.
 4. database.db: فایل دیتابیس SQLite که برای ذخیره سوابق محصولات، مشتریان و سفارشات استفاده می‌شود.
 5. main.py: ارائه رابط کاربری خط فرمان برای تعامل با موجودی، مشتریان و سفارشات.

نحوه استفاده:

 1. فایل main.py را اجرا کنید تا برنامه شروع شود.
 2. با پیروی از دستورالعمل‌های روی صفحه، موجودی، مشتریان و سفارشات را مدیریت کنید.
